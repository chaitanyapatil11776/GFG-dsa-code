<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>s = "Geeks", c = 'e'
<strong>Output:</strong> 2
<strong>Explanation</strong>: The last occurrence of 'e' is at index 2.
</span></pre>
<pre><span style="font-size: 18px;"><strong>Input</strong>: s = "okiyh", c = 'z'
<strong>Output:</strong> -1
<strong>Explanation</strong>: The character 'z' is not present in the string.</span></pre>
