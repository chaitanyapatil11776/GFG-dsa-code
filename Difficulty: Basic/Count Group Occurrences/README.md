<h2><a href="https://www.geeksforgeeks.org/problems/count-the-characters1821/1">Count Group Occurrences</a></h2><h3>Difficulty Level : Difficulty: Basic</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given a string <strong>s</strong>. Count the characters that have ‘k’ number of occurrences. If a character appears consecutively it is counted as 1 occurrence.</span></p>
<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>s = "abc", k = 1
<strong>Output:</strong> 3
<strong>Explanation</strong>: 'a', 'b' and 'c' all have 1 occurrence.</span>
</pre>
<pre><span style="font-size: 18px;"><strong>Input</strong>: s = "geeksforgeeks", k = 2
<strong>Output:</strong> 4
<strong>Explanation</strong>: 'g', 'e', 'k' and 's' have 2 occurrences.</span></pre>
<pre><span style="font-size: 18px;"><strong>Input</strong>: s = "aaaaa", k = 2
<strong>Output:</strong> 0
<strong>Explanation</strong>: There is only one group occurrence of 'a'</span></pre>
